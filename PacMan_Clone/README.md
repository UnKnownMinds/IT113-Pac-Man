# 🟡 PacMan Clone (C + Raylib)

A classic **PacMan game clone** developed in C using the **Raylib graphics library** as part of a first-semester team project.

## 👨‍💻 Team Project

This project was developed by a team of 10 members.
I contributed as the **Team Lead**, coordinating development and implementation.

---

## 🎮 Features

* Classic PacMan gameplay
* Sound effects and background music
* Custom fonts and UI
* Keyboard controls
* Game-over and restart system

---

## 🛠️ Tech Stack

* **Language:** C
* **Graphics Library:** Raylib
* **Assets:** Custom audio & fonts

---

## 📁 Project Structure

```
├── src/            # Source code
├── include/        # Raylib headers
├── lib/            # Raylib libraries
├── resources/      # Fonts and audio files
├── docs/           # Project documentation
└── README.md
```

---

## ⚙️ How to Run

### Option 1: Using Precompiled Binary

Run the `.exe` file (Windows only).

### Option 2: Compile from Source

Make sure Raylib is installed.

```bash
gcc src/main.c -o pacman -lraylib -lopengl32 -lgdi32 -lwinmm
./pacman
```
